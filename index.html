Features

R2 Hold
L2/LT + DPAD ←

Holds down the R2/RT trigger automatically. Perfect for continuous firing with any full auto weapons. For best results is in the Tips & Usage section.

X/A Spam
L2/LT + DPAD ↓

Rapidly presses X/A button very Useful for fast interactions, spamming through menus.

LQ
L2/LT + DPAD →

Slow auto-clicker optimized for the Liquid Divinium / Dr. Monty's Factory menu. Clicks once every 11-12 seconds to be safe so game dont crash.

SQ
L2/LT + DPAD ↑

Rapidly presses SQ/X button for quick actions in-game/menu.

LED Feedback

Gray — All features OFF
Green — R2 Hold ON
Blue — X Spam ON
Purple — LQ Clicker ON
Pink — SQ ON
